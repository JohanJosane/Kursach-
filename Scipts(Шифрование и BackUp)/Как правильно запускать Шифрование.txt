Сам запрос шифрования:
(Не путаем, что после UPDATE надо писать не название базы данных, а название таблицы!!!)


UPDATE ClientProgram
SET Login = CONVERT(varchar(64),
HASHBYTES('MD5',Login),2)

UPDATE ClientProgram
SET Password = CONVERT(varchar(64),
HASHBYTES('MD5',Password),2)

SELECT * FROM ClientProgram

Если Палыч попросит зашифровать файлы в Базе Данных, 
то тебе придеться изменить зашифрованные значения на обычные(то есть Логин 1, Пароль 1).

И потом опять запустить запрос. Так как если не сделать то что выше написано и запустить сразу запрос, 
то запрос будет шифровать уже зашифрованные данные(и ты не сможешь зайти в программу)!!!
